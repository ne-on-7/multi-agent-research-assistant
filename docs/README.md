# docs/

Place the demo recording here as `demo.gif` — the root `README.md` references it
as `![Demo](docs/demo.gif)`.

Recording tips:
- ~10–20 seconds, looped: ask a question → the three agent panels update → the
  cited answer streams in.
- Keep it under ~10 MB (GitHub won't render larger inline). Compress with
  [Gifski](https://gif.ski/) or `ffmpeg`; 1200px wide at ~10 fps is plenty.
